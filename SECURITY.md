# Security Policy

## Supported Versions

Currently, only the latest release of `ui-components` is supported with security updates. 
Since this repository consists of copy-pasteable UI components, security vulnerabilities are generally isolated to build tools (Vite) or animation libraries (GSAP). 

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security is a top priority for our Open Source community. If you discover a security vulnerability in the repository's configuration, build tools, or component logic, please do not open a public issue.

Instead, please responsibly disclose the vulnerability by sending an email to the repository owner or using the "Security Advisory" tab on GitHub to report it privately.

We will endeavor to respond to your report within 48 hours and provide a timeline for a fix.
